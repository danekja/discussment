  • Request
  • Response
  • Security

URI Parameters

  • discussionId: required(integer)

    Discussion ID.

HTTP status code 200

Posts retrieved successfully.

Body

Media type: application/json

Type: array of object

Items: PostDTO

  • id: required(integer)

    Unique ID of the post.

  • text: required(string)

    Text of the post.

  • state: required(one of ENABLED, DISABLED)

    State of the post.

  • level: required(integer)

    Nesting level of the post. Starts from 0, increases by 1 for every reply level.

  • createdBy: required(string)

    Username of the user who authored the post.

  • createdAt: required(datetime-only)

    Datetime when the post was created in UTC timezone.

  • likes: required(integer)

    Number of LIKED reactions attached to the post.

  • dislikes: required(integer)

    Number of DISLIKED reactions attached to the post.

  • parentalPostId: (integer)

    Parental post ID in case this post is a reply.

  • replies: required(array of $recur)

    One level of replies sent to the post.

  • currentUserReaction: (object)

    Reaction of the currently authenticated user. Null if no user is authenticated or the user has not voted.

    • type: required(one of LIKE, DISLIKE)

      Type of a reaction.

    Example:

    {
  "type": "LIKE"
}
